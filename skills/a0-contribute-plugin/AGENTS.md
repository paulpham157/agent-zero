# Plugin Contribution Skill DOX

## Purpose

- Own the workflow for publishing an Agent Zero plugin to the community Plugin Index.
- Keep contribution guidance aligned with current Plugin Hub and `a0-plugins` repository requirements.

## Ownership

- `SKILL.md` owns trigger metadata and the contribution procedure.

## Local Contracts

- Keep repository, `index.yaml`, manifest, license, validation, and PR guidance current.
- Do not hardcode user credentials, personal repository names, or private URLs.
- Keep handoffs to plugin review and management skills accurate.

## Work Guidance

- Update this skill when Plugin Index CI rules, manifest expectations, or submission flow changes.
- Prefer concise command examples that a user can adapt.

## Verification

- Manually read `SKILL.md` for broken links, stale paths, and invalid relative references.
